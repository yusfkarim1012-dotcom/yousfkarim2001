2. PORTABLE ENVIRONMENT SETUP
--------------------------------------------------------
The project is designed to be portable. It uses two main scripts:
- yousf.bat: Sets up the local environment paths (Flutter, Android SDK, JDK).
- setup_portable_env.ps1: Automatically downloads the required SDKs if they are missing.

3. HOW TO START ON A NEW COMPUTER
--------------------------------------------------------
1. Clone the project from GitHub.
2. Open the folder.
3. If the 'devtools' folder is missing, run 'setup_portable_env.ps1' in PowerShell.
4. Run 'yousf.bat' to start working.
5. For AI Agents: "Tell Antigravity to read ANTIGRAVITY_GUIDE.md and prepare the environment."
